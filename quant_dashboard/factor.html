📚 测评方法论

• Rank IC: 因子值排名与未来5日收益率排名的Spearman秩相关系数。

• IR: IC均值 / IC标准差，衡量因子预测的稳定性。

• IC胜率: IC > 0 的日期占比，反映因子方向正确率。

• 单调性: 分组收益与组别的相关性，越高说明因子区分力越强。

• IC稳定性: 后半段IC均值 / 前半段IC均值，检测因子是否在衰减。

• 评级标准:
A: IC>0.05 + IR>0.5 + 胜率>65%
B: IC>0.03 + IR>0.3 + 胜率>55%
C: IC>0.02 + IR>0.2
D: IC>0.01
F: IC≤0.01
